FACEBOOK ADS PERFORMANCE ANALYSIS & INSIGHTS – POWER BI DASHBOARD

Project Overview  
This project analyzes Facebook advertising campaign data using a Power BI dashboard.  
The dashboard provides insights into impressions, clicks, ad spend, conversions, click-through rate (CTR), cost per click (CPC), and audience demographics such as age and gender.  
It helps evaluate campaign effectiveness, audience engagement, and cost efficiency to support data-driven marketing decisions.

Dataset Information  
● Source: Facebook Ads Campaign Dataset  
● Format: CSV / Excel  
● Records: Campaign-level and daily performance data  
● Granularity: Date, Campaign, Age Group, Gender  

Key Columns Used  
● Reporting Start Date – Campaign reporting date  
● Campaign ID / Campaign No – Campaign identifier  
● Impressions – Number of ad views  
● Clicks – Number of ad clicks  
● CTR – Click-through rate  
● Spent – Total ad spend  
● Approved Conversions – Successful conversions  
● CPC – Cost per click  
● Age – Age group of audience  
● Gender – Male / Female  

Objectives of the Project  
● Analyze overall Facebook ads performance  
● Measure impressions, clicks, and conversions  
● Understand CTR across age groups  
● Analyze clicks and CPC by campaign  
● Study audience behavior by age and gender  
● Track click trends over time  

Dashboard Features

Visualizations Used

Tools & Technologies Used

Data Cleaning & Preparation (Step-by-Step)  
● Imported Facebook Ads dataset into Power BI  
● Checked for missing and inconsistent values  
● Converted date and numeric columns  
● Standardized campaign and demographic fields  
● Created calculated measures using DAX  
● Validated filters and visual interactions  

Measures Created  
● Total Impressions  
● Total Clicks  
● Total Spent  
● Approved Conversions  
● CTR  
● Cost per Click (CPC)  

Key Insights :  
● Certain age groups show higher CTR and conversions  
● One campaign contributes the majority of clicks  
● Male audience generates more clicks compared to female  
● Click activity fluctuates across reporting dates  
● Higher CPC campaigns do not always yield higher conversions
